# Handled

Handled is an email relief app.

Core promise:
Your inbox. Handled.

Non-negotiable rule:
The app never sends emails without explicit user approval.

MVP goal:
Build a prototype using fake email data first. No Gmail connection yet. No real AI yet.

Main screens:
1. Home / Today
2. Email Detail
3. Settings

Modes:
- Assist Me: AI suggests, user approves
- Clean My Inbox: app hides clutter
- Handle It For Me: advanced mode for safe automation later

Design:
Clean, calm, minimal, trustworthy.
Use simple language.
